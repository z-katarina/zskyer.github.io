置顶文章方法

  • 参考：解决Hexo文章置顶问题
  • 编辑这个文件：node_modules/hexo-generator-index/lib/generator.js
  • 然后在文章头部的：Front-matter位置加上一个：top: 1000的内容。数值越大，越靠前。

常用插件安装

  • 支持RSS：npm install hexo-generator-feed –save
  • 生成站点地图：npm install hexo-generator-sitemap –save
  • 生成百度站点地图：npm install hexo-generator-baidu-sitemap –save
  • HTML 压缩：npm install hexo-html-minifier –save
  • JavaScript 压缩：npm install hexo-uglify –save
  • CSS 压缩插件：npm install hexo-clean-css –save
  • SEO优化：npm install hexo-generator-seo-friendly-sitemap

域名配置

费域名申请

在校大学生可以申请Github Education提供的免费域名一年（实际还有更多优惠），申请网址 https://education.github.com/pack/ ，因申请人过多，且假的edu邮箱泛滥，故现需要通过提供学生证或者成绩单证明，申请时间一般两到三天。进入 https://nc.me/ ，填写你要注册的域名，加入购物车，选择Github Pages Setup，填写edu邮箱，注册一个namecheap账号，进入https://ap.www.namecheap.com/ ，即可看到已经购买成功的.me域名。

域名参数修改

在namecheap个人页面，点击MANAGE按钮，再选择Advanced DNS。注意，如果是按照我之前方法进行申请的话，此处参数不需要修改。192.30.252.153 和 192.30.252.154 为默认的Github官方提供的两个IP地址。

绑定到Github Pages

在本地博客文件夹的source目录下，新建一个CNAME文件，里面填写你注册的域名，比如我的是zxsky.me，再执行命令 hexo d 上传到github上即可。
