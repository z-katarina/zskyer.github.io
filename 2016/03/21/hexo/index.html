hexo 博客搭建

Hexo
  • Github
  • Hexo

关于Hexo是什么，以及为什么要用Hexo来搭建静态博客在此不多说，重点记录搭建博客的主要过程，后续会把博客的一些更新也记录在此。

环境安装

  • 安装 git
  • 安装 Node.js
  • 安装 Hexo
  • 创建Hexo项目

配置和使用Github

  • 注册一个Github账号
  • 创建一个仓库，仓库命名为xxxx.github.io（xxxx为github用户名）
  • 配置SSH keys

  • 添加SSH key到Github
    打开 C:\Users\Administrator.ssh\id_rsa.pub，复制文件中的内容。
    访问 https://github.com/settings/ssh ，添加新秘钥。

  • 设置用户信息

配置Hexo主题

  • 把本地的博客内容同步到Github上
  • 下载喜欢的Hexo主题，我选择的主题是 yelee
  • 修改配置文件_config.yml
  • 部署到Github上
  • 访问服务器地址进行检查：http://zskyer.github.io

美化Hexo主题

  • yelle主题配置
  • 常用页面添加

  • 多说评论样式修改（旋转头像、浏览器标识等）
    参考原作者博文 http://moxfive.xyz/2015/09/29/duoshuo-style/
    相关下载：duoshuo.css embed.js

  • 自动生成Tag Cloud标签云页面
    参考原作者博文 http://moxfive.xyz/2015/10/25/hexo-tag-cloud/
